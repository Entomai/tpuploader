# TP Uploader Overview

TP Uploader is a streamlined file management and upload utility for Botble CMS. It extends the core media capabilities to handle chunked uploads, specific format processing, and integration with third-party storage providers.

## Key Features

- **Chunked Uploads**: Seamlessly process large files by breaking them into smaller chunks.
- **Enhanced Storage Management**: Fine-grained configuration for how and where files are stored.
- **Media Integrations**: Tightly integrates with the core Botble Media module.
